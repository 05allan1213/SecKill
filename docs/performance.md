# 性能优化

## 已落地优化

- `user-main`、`seckill-main` 的 DB 连接池统一补齐 `SetConnMaxIdleTime(10m)`
- `seckill-main` 的 `SecKillV1`、`SecKillV3` 读取商品改为优先走缓存
- `seckill-main` 的 Redis 预扣库存模型提取统一 key/序列化辅助函数
- 秒杀热路径去掉整段 JSON info 日志，只保留错误日志和必要摘要
- `wrkbench` 统一改为通过网关压测

## 压测命令

```bash
make bench-smoke
```

或直接执行：

```bash
./scripts/bench_smoke.sh
```
